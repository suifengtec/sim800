## 世讯SIM800C模块

世讯旗舰店出售的SIM800C模块,在给定电压是5V时,一直注册不上网络:

```
AT

OK

RDY

+CFUN: 1
AT+CREG?

+CREG: 0,4

OK

+CPIN: READY

Call Ready
AT+CREG?

+CREG: 0,2

OK

SMS Ready
AT+CBC

+CBC: 0,79,4032

OK
AT+CSQ

+CSQ: 20,0

OK
AT+CREG?

+CREG: 0,2

```

CSQ表示网络质量,第一个数字表示信号强度:


CREG表示是否连接到网络:

`0,2`表示正在寻网,`0,5`表示已建立网络连接。

[关于AT命令的说明](https://seeeddoc.github.io/AT_Command_Tester/)
